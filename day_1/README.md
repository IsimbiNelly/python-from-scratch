Day 1: What I Learned in Python

- I learnt how to use print() to display messages.
- I learnt how to write and combine strings.
- I used \n to write text on new lines.
- I learnt how to add comments using #.
- I learnt how to get user input using input().
- I used len() to count the number of characters in a string.
- I learnt that spacing and indentation are important in Python.
- I built a simple Band Name Generator using user input.

SUMMARY
On Day 1, I learned the basics of Python such as printing text, working with strings, taking input from the user, and understanding indentation. I also created a small project to practice what I learned.
